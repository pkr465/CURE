Capabilities
Everything you need for
production-grade C/C++ code
Five deep static adapters, four context layers, LLM-powered code review, human-in-the-loop feedback, and multi-file patch analysis — all in a single pipeline.
⚡

Deep Static Analysis

AST-accurate analysis via CCLS/libclang, Lizard cyclomatic complexity, and Flawfinder CWE-mapped security scanning — not regex heuristics.

🔌

Call Graph & Dead Code

BFS reachability from entry points, fan-in/fan-out analysis, cycle detection, and max call depth. Find unreachable code before it ships.

🛠

LLM Patch Review

Analyses only lines changed by a patch. Runs original vs. patched comparison to identify issues introduced by the change — zero pre-existing noise.

🔎

Context-Aware Analysis

Four context layers — header resolution, constraint validation, call-stack tracing, and parameter validation — give the LLM real codebase context, not just isolated files.

🔧

Auto-Repair Engine

CodebaseFixerAgent reads human feedback from Excel, applies LLM-guided fixes with structural validation, and writes patched files — originals untouched.

📊

Batch Patch Analysis

Analyse entire Perforce changelists, Git diffs, or unified patches. Auto-detects five header formats and runs CodebasePatchAgent per file.

Quick Start
Up and running
in one command
Works on macOS, Linux, and Windows (WSL). The installer handles Python, dependencies, system tools, and environment setup.
Terminal
# One-command setup
chmod +x install.sh && ./install.sh
# Launch the dashboard
./launch.sh
# Launch dashboard + open website
./launch.sh --website
# Run analysis from CLI
python main.py --llm-exclusive --codebase-path /path/to/your/code
